# La-Lluvia-Que-Nadie-Vio
Hay lluvias que caen adentro. Que nadie ve. Que nadie pregunta. Que cargas tú solo, en silencio.  Este es un espacio para esas lluvias. Para ponerles nombre. Para entenderlas. Para sanarlas.  Bienvenido a La lluvia que nadie vió 🌧️  No escribimos poesía. Escribimos lo que tú ya sentías pero no sabías cómo decir.
